Clicakable Components

A menu, tab, button, or other UI widget can directly call up HTML pages but usually can not directly call a component. By importing the data-handler attribute from Sol Components, components become first class citizens, directly callable from a click.

The picker at right uses the data-handler attribute on ordinary buttons to create a tabset that display the PodOS component pos-image.

data-handler supports directly displayhing components based on clicks in menus, tabs, selectors, and buttons.
